# The Forge (new-island params panel) Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Replace the front door's `a new island` direct-roll with **the forge** — a params panel exposing the whole world generator (headline: seed / shape / relief / size / warmth; the full `WorldConfig` behind a fold; a button-triggered minimap preview; ⟳ randomize; generate).

**Architecture:** A pure `forgeArgs(state)` maps the panel's knob values to a `GenArgs` bundle (`{config, shape?, relief?, warm}`) with clamping — testable with no UI. `loadWorld(seed)` gains an optional `gen?: GenArgs` param so the forge (and preview) drive generation directly instead of via URL params. The panel is a new `src/render/forge.ts` DOM module (like `menu.ts`/`picker.ts`); the preview reuses the overview minimap paint. The `a new island` row opens the forge; **generate** does `leaveTitle(); loadWorld(seed, gen); renderer.setMap(map)`.

**Tech Stack:** TypeScript, Vite, Vitest (node env — pure logic TDD'd; DOM/rendering screenshot-verified via `npm run shot`).

## Global Constraints

- **Determinism:** no `Math.random`/`Date.now`/`new Date()` in game/sim logic. The forge's ⟳ **randomize** is a user action and MAY use `randomSeed()` (the one sanctioned `Math.random`, in `main.ts`) for the seed; the generator itself stays seeded/pure.
- **Backdrop invariant still holds:** the forge only ever runs when the player is entering a world (generate → `leaveTitle()` first). The preview must NOT `loadWorld`/persist — it calls `generate()` for a throwaway `WorldMap` and paints it; no `currentSeed`/ledger mutation.
- **Every knob clamps to sane bounds** so no value can hang or white-screen `generate()`; `maxGenerationAttempts` / reroll guards stay in force.
- **`?nomenu=1` deep-link path unchanged:** `loadWorld(seed)` with no `gen` arg behaves byte-for-byte as today (URL `?shape=`/`?warm=` still honored).
- **Art:** codex `:root` tokens; lowercase evocative copy (match `menu.ts`/`picker.ts`).
- **Scope:** button-triggered preview (live-on-drag is deferred); the fine-grain `WorldConfig` knobs sit behind a collapsed fold.
- **Commits:** frequent; end every message with `Co-Authored-By: Claude Opus 4.8 (1M context) <noreply@anthropic.com>`.
- **Verify:** `npm run check` clean, `npx vitest run` green, `npm run build` clean before "done".

The generator surface (from `src/world/config.ts` + `src/world/generate.ts`):
- `generate(seed: number, config?: WorldConfig, shape?: IslandShape, relief?: IslandRelief): WorldMap`
- `SHAPES` (6): highland·twin·ridge·lowland·crescent·skerries; `RELIEFS` (5): rolling·terraced·mesa·gorges·crags; `SHAPE_PHRASE`/`RELIEF_PHRASE` for labels.
- `WorldConfig` fields: width, height, elevationScale, elevationOctaves, moistureScale, moistureOctaves, falloffSharpness, seaLevel, shoreLevel, beachLevel, rockLevel, snowLevel, forestMoisture, marshMoisture, riverCount, riverMinSpringElevation, riverMaxSteps, fallMinDrop, fallMaxCount, fallMinSpacing, craterChance, minLandFraction, minWalkableRegion, maxGenerationAttempts. Defaults in `DEFAULT_CONFIG`.

- [ ] **Step 3: Implement `src/render/forgeArgs.ts`.** Define the interfaces above. `FORGE_BOUNDS` gives `[min, max]` for `width`/`height` (e.g. `[64, 512]`), `warm` `[0, 50000]`, and each `WorldConfig` numeric field a sane range (levels within `[0,1]` and monotonic-friendly, octaves `[1,8]`, scales `[8, 400]`, counts `[0, 40]`, chances `[0,1]`, `minWalkableRegion` `[0, 20000]`). Provide a `clamp(v, lo, hi)` local. `forgeArgs`:

```ts
export function forgeArgs(state: ForgeState): { seed: number; gen: GenArgs } {
  const cfg: WorldConfig = { ...DEFAULT_CONFIG };
  cfg.width = clampBound("width", state.width);
  cfg.height = clampBound("height", state.height);
  for (const [k, v] of Object.entries(state.cfg)) {
    if (v !== undefined) (cfg as any)[k] = clampBound(k as keyof WorldConfig, v as number);
  }
  return {
    seed: state.seed,
    gen: {
      config: cfg,
      shape: state.shape === "roll" ? undefined : state.shape,
      relief: state.relief === "roll" ? undefined : state.relief,
      warm: clamp(state.warm, 0, 50000),
    },
  };
}
```
where `clampBound(field, v)` clamps to `FORGE_BOUNDS[field]` if present, else returns `v`.
